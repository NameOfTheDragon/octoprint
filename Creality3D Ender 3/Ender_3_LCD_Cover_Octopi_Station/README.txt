# Summary

This make is mixed from great "Closed Support for LCD ender 3" from DaniBicho (https://www.thingiverse.com/DaniBicho/about) adding a place for install a raspberry pi 3 and a cooling fan places. Add a bottom layer to put a buck converter too.
I placed a place to rest the usb cable for the printer connecting to the raspberry pi.
I still improving it!

My first make!

UPDATES

Version 3:  Added a slot to pass the lcd ribbon cable inside, a new fan place under raspberry station and 4 slots for XT60 connectors, to let use buck converter.
Thanks for all the support.

Versions 4: The logos are back! Added a version with no rail block and files both in STL and OBJ.

Thanks for the support
