# Summary

This is a case for a Raspberry Pi and Pi Camera for an OctoPi setup.  I designed it to be used with extruded aluminum 2020.  It has a place for a 30mm Fan and fan cover with the OctoPi logo on the fan cover and back of the case, also a Raspberry Pi logo on the front case.  Enjoy!
